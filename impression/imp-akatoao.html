「赤と青のレクイエム」斉藤俊雄・脚本

「すごい！！」　中学校・教師

中学校の国語の教師です。太東中の劇の噂は以前から聞いていて、一度観てみたいと思っていました。それが今日かなってとても嬉しいです。

一言「すごい！！」。久しぶりに感動して涙が出ました。この年齢で、感動して涙を流すのは本当に何十年ぶりです。それだけ太東中の演劇はすごいと思います。本当、素晴らしかった。中学生とは思えない…。

これからもがんばって。ファンになりました。

「先生と生徒達に万歳」　医師

今日再び新しい感動をいただきました。今、練習しているフォーレの「レクイエム」の導入に始まり、いつもながら歯切れよいシーンの転回、アクロバティックなエンターテイメントを忘れぬ楽しさ、BGMの組み合わせの素晴らしさ、そして何よりも感心させられるのはそこに流れる人間愛、さりげない自然保護、エコロジーの大切さへの訴えです。

差別されている人を青い血で表した訴え方、その中に人類すべてに通じる差別問題を描くスケールの大きさを感じました。

そしていつもながらのどんでん返し、劇中劇にもどる手法、素晴らしいと感じました。
指導の先生と生徒達に万歳。

「言葉がでませんでした」　東京ミュージカルアカデミー本科生

あまりにも真正面からメッセージを送られたので、幕が下りた後、言葉がでませんでした。

あのストレートな気持ち。本当に大切なことを、いくつになっても、どの国の人でも、男も女も関係なく持ち続けるのは難しいけれど、人が生きていく上で、人として絶対に必要なことなんだなぁと、改めて考えさせられました。

今のプロの劇団には、本当に感動させられるものが少なくなっていると思います。脚本や演出を含め、この劇にはプロにはない、プロ以上に生き生きとしたもの(役者の若さだけではなく、ピュアな気持ちも含め)を感じます。

舞台装置、音楽、美術もとても美しいものでした。　　　　　　　　　

「中学生だからこそ」　高校生・久喜市在住

演劇部とはいえど、体操部並の運動神経にとにかく驚いた。声の大きさも並外れた大きさだった。

最後の一人が火の中で死んでいってしまうとき、もう涙がこぼれそうだった。

日本全国のどこの中学校に行ってもこんなに素晴らしい劇を見せてくれるところはないと思う。
昨年もここに足を運んだけれど、昨年も今回と同様にとても感動した。

「中学生なのに…」と思う反面、「中学生だからこそ…」できるものがあるんじゃないかと思った。
来年もこの素晴らしい感動を私達に与えてくださいね。　　　　

「今回ほど感動した作品はありません」　高校生・大宮高校

すごく感動しました。私も演劇部に所属しているので、何度も劇というものを見てきましたが、今回ほど感動した作品はあまりありませんでした。

私の所属している演劇部には現在４人の１年生しかいません。そのため他の部活から人を借りて来るという状態で、先日斉藤先生の「遙かなりわが地球」を自主公演で上演させていただきました。もし人数が増えたらこれからも斉藤先生の作品を上演していきたいです。

細々とよくわからない文章を書いてしまってすいません。これからもがんばってください。次回も観に来たいと思ってます。

「中学生かよ！」　高校生・春日部女子高校

はじめがとても印象的。うまい。
反応をもっとしてみてください。
「笑い」うめー！うめーよ！
基本的なことかができすぎてるよ！
「青い月」もっとみしてぇー。
書きたりないよー。

感動したよう。中学生かよ？
おい！どっかの高校よりうまい。
セットすごっ！
私よりうまいよー。マジ！
私も引退公演思い出して、泣いてしまった。

「すごすぎるー！」　高校生・OG

見終わったときの私の状態。
…　…　…　…　…　…　…
…　…　…　…　…　…　…
…　…　…　…　…　…　…
…　…　…　…　…　…　…
(涙が一滴落ちる)　…　…　二滴
…　…　すっ…すごすぎるー！！！
という感じです本当に。
人の心を動かせることほどすごいことはないですね。
変な言い方だけど、こんな時みんな生きてるんだなあって思います。

「これからも応援してます」　中学生・春日部豊野中

最高です。何も言うことがありません。
照明、音響、衣装、声、演技、本当によかったです。
それからバク転などすごかったです。

私も演劇部員の一人なのですが、いつか劇が終わり、みんなと思いっきり泣けるような劇を作っていきたいと思います。
これからも応援してますんで、がんばってください。今日はお疲れさまでした。
